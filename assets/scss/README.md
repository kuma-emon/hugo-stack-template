# assets/scss/

Stack テーマのスタイルを拡張・上書きするカスタム SCSS ファイルを格納しています。

## ファイル一覧

### custom.scss

メインのカスタマイズファイルです。Hugo が自動的に読み込み、テーマの SCSS と合わせてコンパイルします。

含まれるカスタマイズ：

- アフィリエイトリンクのスタイル（ブロック表示・色・点線下線・新タブ表示の視覚的な手がかり）
- 記事一覧レイアウト（アイキャッチ左・メタ情報右の横並び）
- サイドバー幅とアイコン間隔の調整
- フッター区切り線の全幅化
- ダークモード切り替えボタンの外観（sun/moon アイコンの配置）
- コメントセクションのスタイル
- 関連コンテンツのスクロールボタンスタイル

独自のスタイルを追加する場合は、`custom.scss` の末尾に記述してください。

### chroma-light.scss

ライトモード用シンタックスハイライト配色です。VS Code Light+ テーマをベースにしています。

`hugo.toml` で `noClasses = false` を設定している場合、Hugo の Chroma ハイライターがこのファイルを参照します。

ライトモードのシンタックスカラーを変更するには、このファイルの CSS クラス定義を編集してください。

### chroma-dark.scss

ダークモード用シンタックスハイライト配色です。VS Code Dark+ テーマをベースにしています。

ユーザーがダークモードに切り替えると自動的に適用されます。

ダークモードのシンタックスカラーを変更するには、このファイルの CSS クラス定義を編集してください。

## カスタマイズ方法

1. `custom.scss` をエディタで開きます。
2. CSS/SCSS ルールを追加または編集します。
3. `hugo server -D` を起動してライブプレビューで確認します。
4. `main` にプッシュするとデプロイされます。

Hugo の extended 版（このテンプレートで必要）には SCSS コンパイル機能が含まれています。
`deploy.yml` では `extended: true` を指定して CI でも利用できるようにしています。
